Pseudocode
APPROX-VERTEX-COVER(G)
1  C = ∅
2  E' = G.E
3  while E' ≠ ∅
4      let (u,v) be arbitrary edge of E'
5      C = C ∪ {u, v}
6      remove every edge incident on u or v
7  return C
A marginal note. Line 4 says pick any edge — the algorithm is nondeterministic by design. Different choices produce different covers; some lucky, some tight against the bound. Press Shuffle to rerun the same graph under a different edge order. No matter what order you pick, the final |C| is guaranteed ≤ 2·|C*|. That is the whole theorem.
